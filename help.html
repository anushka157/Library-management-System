Overview

Welcome to the Library Management System help page. This system allows you to manage your library by adding and viewing books.

Adding a Book

To add a new book, navigate to the "Add Book" section on the main page. Enter the title and author of the book in the respective fields and click the "Add Book" button.

Viewing Books

The added books will be displayed in the "Book List" section. Each book entry includes the title and author. You can scroll through the list to view all the added books.
